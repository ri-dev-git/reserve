# reserve
The repo makes callw to multiple blcockchains to get the balance and price of the crypto assets for the respective native addresses. 


# Requirements
- node

# Stack used
- nodejs
- mongodb
- express js

# pakages installation
`npm i`

# run locally
`nodemon main.js`
